Portfólio

- Sistema de Gerenciamento de Tarefas Descrição: Um sistema web que permite aos usuários criar, organizar e acompanhar suas tarefas diárias. Inclui recursos como criação de projetos, atribuição de tarefas, prazos e status de conclusão. Tecnologias utilizadas: HTML, CSS, JavaScript, Node.js, Express, MongoDB.

- Aplicativo de E-commerce Descrição: Um aplicativo mobile que permite aos usuários comprar produtos de diferentes categorias, adicionar ao carrinho, fazer o checkout e realizar pagamentos seguros. Também inclui recursos de busca, avaliações de produtos e acompanhamento de pedidos. Tecnologias utilizadas: React Native, Redux, Firebase, RESTful API.

- Plataforma de Aprendizado Online Descrição: Uma plataforma online que oferece cursos em diversas áreas, permitindo que os usuários se inscrevam, assistam às aulas, realizem exercícios e recebam certificados de conclusão. Também inclui recursos como fóruns de discussão e acompanhamento do progresso do aluno. Tecnologias utilizadas: HTML, CSS, JavaScript, Django, PostgreSQL.

- Sistema de Gestão de Projetos Descrição: Um sistema web que ajuda equipes a gerenciar projetos, atribuir tarefas, acompanhar o progresso e colaborar em tempo real. Inclui recursos como gerenciamento de documentos, calendário compartilhado, chat integrado e geração de relatórios. Tecnologias utilizadas: HTML, CSS, JavaScript, Ruby on Rails, MySQL.
